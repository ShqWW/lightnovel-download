

# 轻小说文库 epub格式下载

## 简介
[轻小说文库](https://www.wenku8.net)下载器，只需要根据网址填入小说信息即可下载并打包成EPUB格式小说。

由python语言编写，按Sigil提供的默认EPUB模板打包。

不足：由于轻小说文库没有提供插图在正文中的位置，所以目前只能下载并打包插图到EPUB的图片文件夹，并需要网友们手动指明彩页的页数，对于正文中的黑白插图，需要手动使用EPUB编辑器自行插入到正文中，如果不插入也不会影响正文阅读。目前没有想到针对这一点的解决方法，欢迎网友们提供改进方法。

## 安装

自行使用pip命令， 安装所有[lightmovel.py](https://github.com/ShengqiWang/lightnovel-download/blob/master/lightnovel.py)中所有导入的包。由于包版本变化较大，就不再提供安装命令了，尽量安装最新版即可。
